# 💀 Motivation? Nah, Memevation.

A simple and aesthetic **Motivational Meme Website** built using **HTML5** and **CSS3**. Featuring a clean black-and-white design, bold typography, and a relatable meme that perfectly captures the moment when all your "nonchalance" disappears with a single compliment.

## 🚀 Features

- 🖼️ Responsive meme poster layout
- 🎨 Minimal black & white aesthetic
- ✨ Google Fonts integration (Libre Baskerville)
- 📱 Simple and lightweight design
- 💻 Beginner-friendly HTML & CSS project

## 🛠️ Technologies Used

- HTML5
- CSS3
- Google Fonts (Libre Baskerville)
